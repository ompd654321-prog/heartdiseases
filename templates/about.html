❤️ Heart Disease Prediction Using ANN

About the Project

This project predicts the presence of heart disease using an Artificial Neural Network (ANN). It takes medical inputs such as age, cholesterol, blood pressure, and heart rate to determine whether a person is likely to have heart disease.

Working of the System

  • Dataset is loaded and preprocessed
  • Features and target are separated
  • Data is split into training and testing sets
  • Data is normalized using StandardScaler
  • ANN model is trained
  • Model is evaluated using accuracy
  • Model and scaler are saved

ANN Model Used

This project uses a Multi-Layer Perceptron (MLP):

  • Hidden Layer 1: 64 neurons
  • Hidden Layer 2: 32 neurons
  • Activation Function: ReLU
  • Output: Binary Classification

Technologies Used

  • Python
  • Pandas
  • NumPy
  • Scikit-learn
